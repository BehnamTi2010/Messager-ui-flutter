# 💬 Messenger UI — Flutter

A clean, modern **UI-only** messaging app built with Flutter — inspired by dark-themed chat apps like Telegram/Instagram DMs.

> ⚠️ This project focuses purely on **UI/UX implementation**. No backend, authentication, or real-time messaging logic is included.

## ✨ Features

- 🌙 Dark theme, minimal & modern design
- 📖 Stories row with circular avatars & gradient rings
- 🔍 Search bar for chats
- 💬 Chat list with last message preview
- ⌨️ Live "Typing..." indicator UI
- 🔻 Bottom navigation bar (Calls, Messages, Settings, etc.)
- 📱 Fully responsive layout

---

## 🛠️ Built With

- **Flutter** — cross-platform UI toolkit
- **Dart** — programming language

---

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/BehnamTi2010/<repo-name>.git

# Navigate into the project
cd <repo-name>

# Get dependencies
flutter pub get

# Run the app
flutter run
```

## 📂 Project Structure

```
lib/
 ├── screens/        # App screens (Messages, Chat, etc.)
 ├── widgets/        # Reusable UI components
 └── main.dart        # App entry point
```
